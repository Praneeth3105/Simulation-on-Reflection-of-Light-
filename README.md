# 🌞 Reflection of Light Simulation

An interactive web-based simulation demonstrating the **Reflection of Light** concept using **HTML, CSS, and JavaScript**.  
This project visually explains how light rays behave when they strike different surfaces at varying angles.

---

## 📌 Project Overview
The **Reflection of Light Simulation** is an educational physics project designed to help students understand:
- Incident rays
- Reflected rays
- Angle of incidence
- Angle of reflection
- Effect of different reflecting surfaces

The simulation is fully interactive and runs directly in the browser.

---

## ✨ Features
- Interactive **light ray animation**
- Adjustable **angle of incidence** using a slider
- Multiple reflecting surfaces:
  - Glass
  - Sand
  - Water
- Tooltip-based explanations for objects
- Show / hide light rays
- Reset simulation functionality
- Clean and visually engaging UI
